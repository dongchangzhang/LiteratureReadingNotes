### 如果需要在本地运行源代码

1.修改项目配置文件*

```
source-code-ghz/LiteratureManage/src/com/teamghz/configure/Configure.java
```

该文件中包括：

* 项目中pdf文件存储位置
* 头像位置
* MySQL存储分隔符号
* 头像或pdf文档url开始位置
* url分隔符

以上信息修改需要根据系统是windows或者linux或者mac修改相应位置信息

```
source-code-ghz/LiteratureManage/src/com/teamghz/configure/Configure.java
```
